try_path(path, get_visited=False)[source]¶

Path is tried from entry point, step by step. Course stops either as soon as a wall is met, or it returns to an already visited cell, or reaches the exit. Successful steps are steps until stop.

Parameters:
  • path (an sequence (actually a string but could be list) of DIRECTIONS) – a sequence of DIRECTIONS corresponding to moves.
  • get_visited (boolean) – if True, returns
Returns:

a tuple containing number of successful steps using path and manhattan distance between reached cell and exit

(int, int)
